Markdown介紹

Posted on 2018-04-27 | In Hexo

Markdown is a text-to-HTML conversion tool for web writers. Markdown allows you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid XHTML (or HTML). by John Gruber

前言

1. John Gruber?

John Gruber是發明markdown的人

2. Markdown?

markdown是一種輕量級標記式語言，同時也是由Gruber撰寫的用來把text轉換成HTML的perl程式

常用指令

1. Headings

# Heading level1

Heading level1

## Heading level2

Heading level2

### Heading level3

Heading level3

#### Heading level4

Heading level4

##### Heading level5
Heading level5
###### Heading level6
Heading level6

2. Emphasis

**bold** text

bold text

*italic* text

italic text

***bold and italic*** text

bold and italic text

3. Blockquotes

>The paragraph is a block

The paragraph is a block

4. List

- First item
- Second item
- Third item

  • First item

  • Second item

  • Third item

5. Code block

只要用至少一個Tab縮進去即可～

<<!DOCTYPE html>
<html>
    <head>
        <title></title>
    </head>
</html>

6. Links

My favorite web is [ABC](https://ABC.com)

My favorite web is ABC

7. URLs and Email

<https://ABC.com>
<ABC@gmail.com>

https://ABC.com

ABC@gmail.com

8. Images

![We bare bears](https://ppt.cc/fftcMx)

9. Font color

<font color="red">紅色的字</font>

紅色的字

10. Change background-color of font

<span style="background-color:pink">some **pink** text</span>

some pink text

11. Note label
